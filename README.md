# ⚡ SKITMC | OVERDRIVE

**SKITMC | OVERDRIVE** is a high-performance, industrial-themed incremental clicker game. Step into the role of a Grid Operator, managing a high-voltage energy core to power the world. Harness raw energy through manual input and massive grid infrastructure to reach Tier-7 energy production.

## 🚀 Key Features

* **Dual-Input Generation:** Use your **Mouse** or mash the **Spacebar** to generate Watts.
* **Dynamic Battery HUD:** A segmented XP bar that visually charges as you level up your unit.
* **Massive Upgrade Catalog:** 13+ unique subsystems ranging from simple **Lemon Batteries** to reality-bending **Void Siphons**.
* **System Overclock (Rebirth):** Reset your grid infrastructure for a permanent **3x Global Multiplier** per prestige level.
* **Smart Number Formatting:** Clean UI that scales from Watts to **Quadrillions (Qi)** without breaking the layout.
* **Zero-Loss Persistence:** Automatic local storage saves your progress every 5 seconds.

## 🕹️ Controls

| Action | Control |
| :--- | :--- |
| **Generate Energy** | Left Click / Spacebar |
| **Purchase Upgrade** | Click Item in Shop |
| **Reset Data** | "Wipe Data" in Footer |
| **Prestige** | Click "System Overclock" (Unlocks at Lvl 15) |

## 🛠️ Technical Stack

* **Frontend:** HTML5, CSS3 (Custom Properties / Flexbox)
* **Logic:** Vanilla JavaScript (ES6+)
* **Storage:** Browser `localStorage` API
* **Graphics:** Procedural CSS Glows & Asset-based Core Rendering

---

## 📈 Grid Subsystems (The Shop)

### Manual Overclocking
* **Manual Crank:** Simple physical generation.
* **Copper Coil:** Increases touch conductivity and PPC.
* **Static Friction:** High-cost, high-reward manual multipliers.

### Infrastructure (Auto-Income)
* **Tier 1:** Lemon Battery, Static Carpet.
* **Tier 2:** Solar Arrays, Wind Blades.
* **Tier 3:** Hydro Turbines, Coal Plants.
* **Endgame:** Fusion Cores, Antimatter Coils, Dyson Swarms.

## 💾 Installation & Local Development

1.  Clone the repository or download the `index.html`.
2.  Ensure your `assets/` folder contains your `clicker.png`.
3.  Open `index.html` in any modern web browser.

---

> **SYSTEM NOTE:** Always ensure your core temperature remains stable. Excessive mashing of the Spacebar may result in localized "Surge" events.
